아이콘 파일 안내
================

이 폴더에 다음 파일이 필요합니다:

1. icon.ico - 프로그램 아이콘 (256x256, 128x128, 64x64, 48x48, 32x32, 16x16 포함)

아이콘 생성 방법:
-----------------

방법 1: 온라인 변환기 사용
- https://convertico.com/
- https://icoconvert.com/
- PNG 이미지를 업로드하여 ICO로 변환

방법 2: ImageMagick 사용 (명령줄)
> magick convert icon.png -define icon:auto-resize=256,128,64,48,32,16 icon.ico

방법 3: GIMP 사용
1. PNG 파일을 GIMP로 열기
2. 이미지 > 이미지 크기 조정 (256x256)
3. 파일 > 다른 이름으로 내보내기 > .ico 확장자 선택
4. 여러 크기 선택하여 저장

권장 디자인:
-----------
- 주식 차트 또는 상승 화살표 이미지
- 선명한 색상 (파란색, 녹색 권장)
- 간단하고 인식하기 쉬운 디자인
